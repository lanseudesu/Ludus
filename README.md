# Ludus

Compiler Design project in partial fulfillment of the requirements for the **Automata Theory and Formal Languages** course.

## Overview

Ludus is a custom programming language and compiler developed as a Compiler Design project in partial fulfillment of the requirements for the Automata Theory and Formal Languages course.

The project demonstrates the fundamental phases of compilation, including lexical analysis, syntax analysis, semantic analysis, Abstract Syntax Tree (AST) generation, and interpretation/execution.

## Features

- Custom programming language syntax
- Lexical analysis (tokenization)
- Syntax analysis using LL(1) parsing
- Abstract Syntax Tree (AST) generation
- Semantic analysis and type checking
- Scope management
- Functions with parameters and default values
- Arrays and multidimensional arrays
- Conditional statements (`if`, `elif`, `else`)
- Loops
- Struct support
- Runtime interpreter
- Graphical user interface using Eel
